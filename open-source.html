오픈소스 라이선스

InhouseMaker 앱은 일부 오픈소스 라이브러리를 사용합니다. 각 라이브러리는 해당 라이선스 조건에 따라 사용됩니다.

1. 안내

아래 목록은 iOS 앱에서 자주 사용하는 대표 라이브러리를 기준으로 정리한 예시입니다. 실제 사용 라이브러리와 세부 버전은 앱 버전에 따라 일부 추가, 변경 또는 제외될 수 있습니다.

  • 실제 사용 라이브러리는 앱 버전, 배포 환경, 기능 구성에 따라 달라질 수 있습니다.
  • 각 오픈소스의 저작권은 원 저작자 및 기여자에게 있습니다.
  • 정확한 라이선스 전문과 고지 의무는 각 공식 저장소의 LICENSE 또는 NOTICE 파일을 함께 참고해 주세요.

2. 대표 라이브러리 목록

현재 프로젝트에 맞게 추후 수정할 수 있도록, 앱 안내 페이지 기준으로 읽기 쉬운 형태로 정리했습니다.

코드 기반 화면 구성에서 Auto Layout 제약을 간결하게 작성할 때 자주 사용하는 라이브러리입니다.

3. 추가 안내

GoogleSignIn, AppAuth 계열과 같이 로그인 또는 인증 흐름에 따라 선택적으로 포함되는 라이브러리는 실제 배포 구성에 맞춰 별도 항목으로 추가 표기할 수 있습니다.

정식 고지용 문서를 운영할 때에는 실제 프로젝트의 의존성 목록과 배포 버전을 기준으로 최종 점검해 주세요.
